This folder contains data for the project "Bumble bee species display contrasting phenological responses to climate variation" by Kyle C. Elshoff, Sylvana R. Finn, Erin D. Treanore, Leif. L. Richardson, and Elizabeth E. Crone.

Here's how to navigate this folder:

"data" contains both the raw data and cleaned data used for analyses.
"scripts" contains the R scripts used to process and analyze the data as well as create figures and tables.
"outputs contains analysis outputs (i.e., tables and RDS files with statistical outputs).
"figs_tables" contains the outputs of code that generate figures and tables (or their precursors) that appear in the manuscript or appendix.
